touhou-music-demo
=================

SFML based touhou music demo program

Compile under linux:
--------------------

You need the folowing libraries:
* libsfml-2.1
* sndfile

Clone the repository

    # git clone 'https://github.com/dsonck92/touhou-music-demo.git'
    
Init the repository

    # ./init-repository.sh

Run CMake:

    # cmake .

in the directory and then

    # make
